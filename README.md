# AI Recipe Generator 🍳🤖

## Project Overview
The AI Recipe Generator is a simple web application that creates recipes based on the ingredients provided by the user. Instead of searching online for recipes, users can enter the ingredients they already have at home and the system will generate a recipe suggestion using AI.

The goal of this project is to demonstrate how AI can be integrated with a basic web interface to make everyday tasks easier, such as cooking with available ingredients and reducing food waste.

---

## Features
- Generate recipes using ingredients entered by the user
- Simple and easy-to-use interface
- AI-powered recipe suggestions
- Quick and responsive design
- Helps reduce food waste by using ingredients already available

---

## Technologies Used
- HTML – Structure of the website
- CSS – Styling and layout
- JavaScript – Application logic and API requests
- AI API – Used to generate recipes based on ingredients

---

## How the Application Works
1. The user enters a list of ingredients in the input field.
2. The application sends this information to an AI model.
3. The AI processes the ingredients and generates a recipe suggestion.
4. The generated recipe is displayed on the screen for the user.
